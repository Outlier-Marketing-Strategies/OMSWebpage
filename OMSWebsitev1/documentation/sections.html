A section is represented by a div with a section class. Arrange them, add one, remove one or even, you can create your owns.

Helper Class

Add class bg-black to a section for black background color , or bg-white bg-bright for white background color.

Fullpage effects

This website uses fullpage js v2 to make full screen slide by slide scrolling effects

In your html file (such as index-fullscreen.html), to enable the fullpage.js plugin add class fullpage-container to the main tag page-main and remove class scrollpage-container.

Important : A section must have an attribute called data-section (a section-name) which represents the id of this section (after JavaScript processing). It must be unique per HTML file.

Section background

Section background is located under the or div with classes section-footer scrolldown. Place it at then end of the section-wrapper or before the ending tag of the section.

Add class bg-pos-right to align it right or bg-pos-leftto align it left.

Footer - Scrolldown

Section footer is located under the footer or div with classes section-footer scrolldown. Place it at then end of the section-wrapper or before the ending tag of the section.
